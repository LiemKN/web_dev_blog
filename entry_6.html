JSON

JSON stands for JavaScript Object Notation and is a format for storing and transporting data. It is used to send data from a server to a webpage and is known as "self-describing" and easy to use. JSON is built on two structures. First, a collection of name value pairs called an object in JSON, which may be known as a record, dictionary, hash table, etc in other languages. Second is an ordered list of values commonly known as an array. The following is an example of JSON objects being put in a JSON array. 

            {
              "employees":[
                {"firstName":"John", "lastName":"Doe"},
                {"firstName":"Anna", "lastName":"Smith"},
                {"firstName":"Peter", "lastName":"Jones"}
              ]
            }

To use JSON text in JavaScript, create a JavaScript string containing JSON syntax. 

          var text = '{ "employees" : [' +
            '{ "firstName":"John" , "lastName":"Doe" },' +
            '{ "firstName":"Anna" , "lastName":"Smith" },' +
            '{ "firstName":"Peter" , "lastName":"Jones" } ]}';

Next, use the JavaScript function, JSON.parse(), to convert the previously made string into a JavaScript object. 

          var obj = JSON.parse(text);

You can now use the newly made JavaScript object anywhere necessary within your JavaScript.
